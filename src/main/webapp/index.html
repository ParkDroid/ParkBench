ParkBench API

About

What is this API for?

This API is for access to data on Disney parks, including resorts, parks, hotels, dining, and attractions.

How much data does this API currently have?

At present, this API only has data for Disneyland Resort, but all six worldwide resorts are coming soon! Most resources have GPS coordinates, opening date, and official name.

Future Plans

Resources will soon have more information (type of dining for dining, ride length for attractions). Shopping districts like Downtwon Disney and Disney Sprins will be added as resources soon. Other data planned includes character meets, snacks, transportation, and live wait times.

Can I contribute to this API?

Yes, on Github!

Example Requests

Disneyland Resort

GET parkbenchapi.com/api/resorts/disneyland-resort

Disneyland Park

GET parkbenchapi.com/api/resorts/disneyland-resort/parks/disneyland-park

Disneyland Hotel

GET parkbenchapi.com/api/resorts/disneyland-resort/hotels/disneyland-hotel

Trader Sam's Enchanted Tiki Bar at the Disneyland Hotel

GET parkbenchapi.com/api/resorts/disneyland-resort/hotels/disneyland-hotel/dining/trader-sams-enchanted-tiki-bar

Blue Bayou Restaurant at Disneyland Park

GET parkbenchapi.com/api/resorts/disneyland-resort/parks/disneyland-park/dining/blue-bayou-restaurant

Jungle Cruise at Disneyland Park

GET parkbenchapi.com/api/resorts/disneyland-resort/parks/disneyland-park/attractions/jungle-cruise

Resources

Resorts

A Resort represents one of the six Disney resorts in the world.

Requests

Get a list of resorts GET parkbenchapi.com/api/resorts
Get a specific resort GET parkbenchapi.com/api/resorts/{resortID}

Response

Name Description Type
id The identifier for this resort String
name The official name for this resort String
city The city in which this resort is located String
state The state or province in which this resort is located String
country The country in which this resort is located String
longitude The longitude of the geographic coordinates of this resort double
latitude The latitude of the geographic coordinates of this resort double
dateOpened The opening date of this resort in yyyy-mm-dd format String
links List of links to parent resources and sub-resources List of Linked Resources

Parks

A Park represents a theme park within on of the worldwide resorts.

Requests

Get a list of parks within a specific resort GET parkbenchapi.com/api/resorts/{resortID}/parks
Get a specific park GET parkbenchapi.com/api/resorts/{resortID}/parks/{parkID}

Response

latitude The latitude of the geographic coordinates of this park double
